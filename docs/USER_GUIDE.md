# Blue Plaques Map - User Guide

Welcome to the Blue Plaques Map! This guide will help you discover and explore Johannesburg's heritage sites.

## What Are Blue Plaques?

Blue plaques are commemorative markers placed on buildings and sites of historical significance. They honor important people, events, and places that shaped Johannesburg's rich history.

This interactive map features **386 heritage blue plaques** across Johannesburg, from historic homes and churches to sites of political significance and cultural landmarks.

---

## Getting Started

### Opening the Map

1. Visit the application URL in your web browser
2. The map loads centered on Johannesburg
3. All 386 plaques appear as markers on the map

**Tip**: The map works best on modern browsers (Chrome, Firefox, Safari, Edge)

---

## Browsing Plaques

### Viewing the Map

- **Zoom In/Out**: Use the `+` and `-` buttons in the bottom-right corner, or pinch on mobile
- **Pan**: Click and drag the map, or swipe on mobile
- **Markers**: Each blue marker represents a heritage plaque

### Preview a Plaque

**Hover over any marker** (or tap on mobile) to see a quick preview:
- Thumbnail image of the plaque
- Plaque title

### View Full Details

**Click any marker** to open a popup with complete information:
- **Title**: Name of the heritage site
- **Address**: Physical location
- **Description**: Historical significance and details
- **Categories**: Tags like "Homes, Mansions" or "Military, South African War"
- **Images**: Thumbnail gallery (click any image to view full-size)
- **Heritage Portal Link**: Visit the official Heritage Portal page for more information

---

## Searching for Plaques

### Using the Search Box

1. Click the **hamburger menu** (☰) in the top-left corner
2. Type in the **search box** at the top of the sidebar
3. The map updates in real-time to show matching plaques

**Search covers**:
- Plaque titles
- Descriptions
- Addresses

**Example searches**:
- "Tutu" - Find Desmond Tutu's house
- "Parktown" - Find all plaques in Parktown
- "church" - Find all church-related plaques

**Tip**: Search is case-insensitive and matches partial words

## Filtering by Category

### Available Categories

The map includes plaques in these categories:

- Military, South African War
- Architects, People
- Railways
- Johannesburg Centenary
- Churches, Religious Buildings

### How to Filter

1. Click the **hamburger menu** (☰)
2. Scroll down to the **Filter by Category** section
3. **Uncheck** categories you want to hide
4. The map updates immediately

### Bulk Actions

- **Select All**: Shows all categories (default)
- **Clear All**: Hides all categories (useful for starting fresh)

**Tip**: Combine search and category filters for precise results

## Finding Nearby Plaques

### Using "Find Near Me"

1. Click the **hamburger menu** (☰)
2. Click the **📍 Find Near Me** button
3. Allow location access when prompted
4. The map:
   - Centers on your current location
   - Shows a **blue dot** marking your position
   - Zooms to show nearby plaques

**Requirements**:
- Location services enabled on your device
- Browser permission granted

**Privacy**: Your location is only used temporarily and not stored or shared.

## Viewing Images

### Image Lightbox

When you click a plaque marker, you'll see thumbnail images in the popup. Click any thumbnail to open the **full-screen lightbox**.

### Lightbox Controls

**Mouse/Trackpad**:
- **Previous Image**: Click the left arrow (◄)
- **Next Image**: Click the right arrow (►)
- **Close**: Click the X in the top-right corner

**Keyboard**:
- **Previous Image**: Left arrow key (←)
- **Next Image**: Right arrow key (→)
- **Close**: Escape key (Esc)

**Mobile**:
- **Previous/Next**: Tap the arrow buttons
- **Close**: Tap the X button

**Tip**: Images include captions describing what you're viewing

## Reporting Issues

### How to Report a Plaque

If you notice incorrect information or issues with a plaque:

1. Click the plaque marker to open the popup
2. Click the **⚠️ warning icon** in the bottom-right of the popup
3. Confirm you want to report the plaque
4. The plaque is flagged for review by administrators

**What to report**:
- Incorrect location
- Wrong or outdated information
- Broken images
- Inappropriate content

**Note**: Reporting is anonymous and helps improve the map's accuracy.

## Troubleshooting

### Map Not Loading

**Problem**: Blank screen or map doesn't appear

**Solutions**:
1. Refresh the page (F5 or Cmd+R)
2. Check your internet connection
3. Try a different browser
4. Clear browser cache

### Images Not Showing

**Problem**: Broken image icons or missing photos

**Solutions**:
1. Wait a few seconds for images to load
2. Check your internet connection
3. Some older plaques may not have photos available

### "Find Near Me" Not Working

**Problem**: Location button doesn't work

**Solutions**:
1. Enable location services on your device
2. Grant browser permission when prompted
3. Check if location services are blocked in browser settings
4. Try using HTTPS (some browsers require secure connection)

### Search Returns No Results

**Problem**: Search shows no plaques

**Solutions**:
1. Check spelling
2. Try broader search terms
3. Make sure at least one category is selected
4. Click "Select All" to reset filters

## Frequently Asked Questions

### How many plaques are on the map?

Currently **386 heritage blue plaques** are featured.

### Can I download the map?

The map requires an internet connection to load map tiles and plaque data. Offline access is not currently available.

### Can I suggest new plaques?

Admin users can add new plaques through the admin interface. Contact the site administrator to suggest additions.

### Are all Johannesburg plaques included?

This map includes plaques documented in the Heritage Portal database. New plaques are added as they're discovered and verified.

### Can I share a specific plaque?

Currently, there's no direct link-sharing feature. You can share the main map URL and provide search instructions.

### Is this map official?

This is a community-driven project sourcing data from the Heritage Portal. For official heritage information, visit the Heritage Portal directly via the links in each plaque popup.

### Can I use this map for research?

Yes! The map is a valuable research tool. For academic citations, reference both this map and the original Heritage Portal sources.

### How often is the map updated?

The map is updated as new plaques are discovered and verified. Report any issues to help keep information accurate.
